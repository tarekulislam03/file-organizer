The script will organize certain file types into certain folders. (eg. .pdf --> PDFs folder)
It uses Python’s built-in 'os' and 'shutil' modules.
No external dependencies are required.
Safe to run — you can inspect the source code yourself.

-- Steps --
1. Double click the terminal.bat file and write this command --> python
2. It will redirect to the python download page, download and install it..
# if python is installed already ignore step no 1 and 2. 
3. Again open terminal.bat and write or paste this command --> python main.py 
4. It will automatically organize the files

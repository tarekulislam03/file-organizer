The script will organize certain file types into certain folders. (eg. .pdf --> PDFs folder)
It uses Python’s built-in 'os' and 'shutil' modules.
No external dependencies are required.
Safe to run — you can inspect the source code yourself.

-- Steps --
1. Download python and install (must check ADD to Path when installing) from official website --> https://www.python.org/downloads/
2. Double click on terminal.bat file
3. Write or paste this command --> python main.py
4. It will automatically organize the files
